# etl_mini_pipeline
End-to-end Sales ETL pipeline built using Python and PostgreSQL. This project extracts raw CSV data, performs data cleaning and transformation, and loads structured data into a PostgreSQL database for analytics, where advanced SQL analytics queries were executed on the cleaned dataset to generate business insights.

Steps:
1. Extract raw sales data from CSV
2. Transform and clean data using Python
3. Load cleaned data into PostgreSQL
4. Run SQL analytics queries
